
# Maze Generator v11.2 (Black & White, Rectangular Only)

- Perfect-maze generator (DFS backtracker) with unique solution.
- Start/End on opposite edges, carved into interior (always solvable).
- Labels placed *outside* the maze, horizontal, with padding.
- Output: 2-page PDF (puzzle + solution), 300 DPI.
- Maze shapes coming soon
- CLI:
  ```bash
  python maze_generator.py --difficulty hard --wall_thickness 2
  ```
